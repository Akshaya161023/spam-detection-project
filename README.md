# 📩 Spam Detection ML Project

## 📌 Overview
This project is a Machine Learning-based Spam Detection system that classifies messages as Spam or Not Spam using NLP techniques.

## 🚀 Features
- Text preprocessing using NLP
- Machine Learning model (Naive Bayes / Logistic Regression)
- Streamlit web interface
- Real-time spam prediction

## 🧠 Model Used
- Multinomial Naive Bayes (Best for text classification)

## 📂 Dataset
- SMS Spam Collection Dataset
